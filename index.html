Minecraft Mod:

Tools ReAmped

Created:20JUL2015

Coding Language:Java

This mod adds new versions of Diamond Tools that are intended to create an easier or more fun experience for minecrafters. Each Full version Tool, when holding, will give the user a specific boost such as speed boost, strength boost, resistance, etc. Amped Tools are even more powerful versions but with some drawbacks(debuffs) to make it not so easy and switching tools a necessary strategy.
